CAMELS US: Hourly USGS discharge observations and NLDAS forcings

This dataset contains hourly discharge observations and NLDAS forcings for 516 CAMELS US basins.

The dataset was compiled for Rainfall-Runoff Prediction at Multiple Timescales with a Single Long Short-Term Memory Network and contains hourly USGS discharge observations as well as hourly, basin averaged NLDAS forcings for 516 CAMELS US basins. For more details on the data, we refer to the paper linked below.
